# To-Do List Web Application

## 📌 Description

A simple task management web application built using HTML, CSS, and JavaScript.

## 🚀 Features

* Add tasks
* Delete tasks
* Mark tasks as completed
* Data stored using localStorage

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
